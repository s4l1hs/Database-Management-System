About the Project

This project is built on World Development Indicators (WDI) data and is designed to support cross-country comparison across multiple domains (health, energy, freshwater, greenhouse gases, sustainability). The application focuses on normalized indicator reporting, time-series trend computation, region-level aggregation, and transparent visualization of country- and region-level patterns.

Key Features

  • Country and region-level aggregation of indicators across multiple domains.
  • Time-series trend analysis with explicit handling of missing and irregular data.
  • Interactive maps and tables enabling drill-down from global to country-year detail.
  • Consistent units, normalized scales, and comparable metrics across countries.

Data & Methodology

  • Data source: World Development Indicators (WDI), using the latest available country-year records for each indicator.
  • Normalization: indicators are cleaned and aligned by country codes, domains, and units to allow cross-country comparison.
  • Missing data: missing values are left un-imputed and explicitly marked in the UI; trends are computed only where sufficient history exists.
  • Trend computation: percentage and absolute changes are calculated between the earliest and latest available observations per country and indicator.
