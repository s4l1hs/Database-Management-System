  • Snapshot Explorer compares countries for a selected indicator and year using global and regional averages.
  • Global Average is computed using only countries with available data for the selected year.
  • Top 10 shows the highest values for the selected indicator/year; Bottom 10 shows the lowest values.
  • Regional Averages aggregates values by region (as stored in the countries table).
  • Units are shown under the selected indicator (Snapshot) and in the Unit column for each record.
  • Missing values are not included in averages/ranks and may appear as “-” in the Records table.
  • Records can be filtered by country/indicator/year and are paginated (50 records per page).
